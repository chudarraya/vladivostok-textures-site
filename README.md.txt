# Digital_Art_Archive_VDK

Интерактивный 3D-архив текстур Владивостока.

## 🚀 Развёртывание

### 1. GitHub Pages
1. Создай репозиторий на GitHub
2. Загрузи все файлы в корень репозитория
3. В Settings → Pages → выбери ветку `main` и папку `/ (root)`
4. Сайт будет доступен по `https://твой-ник.github.io/репозиторий/`

### 2. Tilda
1. На странице Tilda добавь HTML-блок (T123)
2. Вставь код:
```html
<iframe 
  src="https://твой-ник.github.io/репозиторий/" 
  style="width:100%;height:100vh;border:none;"
  allow="autoplay;fullscreen"
  loading="lazy">
</iframe>
